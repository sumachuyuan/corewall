# 关于 Corewall

> **Corewall：把网页变成你的 macOS 动态壁纸**

App 正在 App Store 审核中，目前可以通过 [TestFlight](https://testflight.apple.com/join/8hBCbycu) 体验。

### 系统要求
- **macOS 14.0 (Sonoma)** 或更高版本
- 支持 **Apple Silicon** / **Intel** 芯片

## 为什么做这个？

macOS 的动态壁纸一直很漂亮，但选择有限。我想，能不能把任意网页、视频变成桌面壁纸？于是做了 Corewall——一个让桌面"活起来"的工具。

## 核心功能

### 🌐 网页壁纸
这是我最喜欢的功能。粘贴任意网址，就能把整个网站变成你的桌面。
- **动态十足**：比如粒子动画网站，桌面上会飘着流动的光点。
- **沉浸体验**：3D 场景网页，桌面直接变成可交互的虚拟空间。
- **实时数据**：实时数据可视化网站，让桌面显示动态图表。
- **鼠标交互**：不只是静态画面，你还可以用鼠标与网页内容进行交互。

### 🎬 视频壁纸
支持 4K 视频直接铺满桌面，且支持音频播放。
- **极致性能**：采用 AVFoundation 渲染引擎深度优化，CPU 占用极低，不会让你的电脑发热。

### 📁 本地文件
除了在线资源，你也可以导入自己的视频、图片。
- **简单易用**：直接拖拽文件进应用即可完成设置。

### 🎰 Lucky Drop
内置了精选壁纸库，涵盖了 ACG、赛博朋克、自然风景等多种分类。
- **盲盒惊喜**：内置了一个"抽奖"功能，可以随机解锁壁纸，每次切换都有新发现。
